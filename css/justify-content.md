# justify-contentについて

## 使い方

`justify-content`プロパティはフレックスアイテムの主軸方向の揃え位置を指定する。<br />


|                   |                                        |
|:------------------|:---------------------------------------|
| 初期値            | flex-start                             |
| 継承              | なし                                   |
| 適用される要素    | フレックスコンテナ                     |
| モジュール        | CSS Flexible Box Layout Module Level 1 |
